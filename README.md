# framer_motion_pages

## 프로젝트 개요

이 프로젝트는 Next.js를 사용하여 개발된 웹 애플리케이션입니다. 스크롤 애니메이션과 반응형 디자인을 특징으로 하며, TypeScript를 사용하여 개발되었습니다.

## 기술 스택

- Next.js
- React
- TypeScript
- Tailwind CSS

## 주요 기능

- 반응형 레이아웃
- 스크롤 애니메이션
- SEO 최적화
